# **App Name**: AstuSphere

## Core Features:

- AI Study Companion Tool: An intelligent study tool that analyzes past exams and curriculum data to provide context-aware solutions and simplified explanations.
- Google Play Showcase Hero: A high-conversion landing section with smooth phone mockups, floating gradients, and download statistics reflecting a premium store experience.
- Live Centralized Archive: An interactive, searchable repository for organized past exams and courses using high-performance filtering systems.
- Dynamic GPA Tracker: A persistent database integration that allows students to store and calculate academic performance in a secure, intuitive dashboard.
- Integrated Code Runner: A feature-rich environment allowing computer science students to run and test code snippets directly within the application context.
- Smart Campus Notifications: Real-time announcement hub for students to stay updated on critical academic events and administrative portals.
- Review Carousel & FAQ: Social proof and trust components utilizing animated carousels and smooth-opening accordion animations.

## Style Guidelines:

- Primary color: Modern Vibrant Green (#34A853) signifying growth and clarity, as requested by the user.
- Background color: A refreshing light desaturated green tint (#F3F8F4) that keeps the design feeling clean and open.
- Accent color: A lively Lime Green (#57D12F), chosen for high visibility and to be analogous to the primary shade.
- Font recommendation: 'Poppins' for headlines and 'Inter' for body text. This pairing offers a professional yet highly readable tech-modern look. Note: currently only Google Fonts are supported.
- Lucide icons used for consistent, stroke-based imagery that feels part of a cohesive Google-inspired design system.
- Utilizes responsive glassmorphism containers and wide rounded corners (28px) to emulate the premium look of a native OS app page.
- Extensive use of Framer Motion for scroll-reveals, parallax background effects, and floating mockup animations to ensure a 'Play Store' premium feel.
